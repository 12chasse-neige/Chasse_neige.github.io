(b) 求证：若两事件间的间隔是类空间隔，则

i. 两事件之间无法建立联系.

证明：

两事件之间的间隔 s2=(x2−x1)2−(ct2−ct1)2>0为类空间隔，这意味着空间距离超过光在该时间间隔内能传播的距离，因此任何物质粒子或电磁波都无法在两事件之间传播，从而无法建立因果联系。

ii. 事件之间的时间次序不确定，存在一个参考系，在其中两事件同时发生.

证明：

考虑两事件在惯性系 S 中的坐标为 (x1,t1) 和 (x2,t2)，类空间隔满足 (x2−x1)2>(ct2−ct1)2。通过洛伦兹变换到另一惯性系 S'，变换关系为：

(5)x′=γ(x−vt)
(6)t′=γ(t−vxc2)

其中 γ=11−v2c2

两事件在 S' 系中的时间差为：

(7)Δt′=γ(Δt−vΔxc2)

要使 Δt′=0，需满足：

(8)Δt=vΔxc2

由于 Δx 和 Δt 满足类空间隔条件，总能找到合适的 v 使得上式成立，即存在一个参考系 S'，在其中两事件同时发生。同时，由于类空间隔下不同参考系中时间次序可能改变，因此时间次序不确定。

iii. 事件之间的空间次序不会颠倒，不存在一个参考系，在其中两事件在同一地点发生.

证明：

对于空间次序，考虑两事件在 S 系中的空间顺序为 x1<x2。在 S' 系中，空间坐标差为：

(9)Δx′=γ(Δx−vΔt)

若存在参考系使 Δx′=0，则需：

(10)Δx=vΔt

代入类空间隔条件 Δx2>(cΔt)2，得：

(11)v2Δt2>c2Δt2⟹v2>c2

这与 v 的物理意义（小于光速）矛盾，因此不存在这样的参考系。同时，由于 Δx′ 的符号由 Δx−vΔt 决定，而类空间隔下无法使 Δx′ 改变符号，因此空间次序不会颠倒。

(c) 考虑三个事件，若已知第一个事件和第二个事件之间的间隔及第二个事件和第三个事件之间的间隔类型，能否知道第一个事件和第三个事件之间的间隔类型？请证明。

不能，构造反例如下：

考虑三个事件 A、B、C，A 和 B 为类空间隔，B 和 C 也为类空间隔，但 A 和 C 可能是类时间隔、类空间隔或类光间隔，具体取决于它们的坐标关系，以下讨论采取自然单位制。

设 A 在原点 (0,0)，B 在 (2,1)，C 在 (4,0)。A 和 B 的间隔 sAB2=4−1=3>0（类空），B 和 C 的间隔 sBC2=4−1=3>0（类空），A 和 C 的间隔 sAC2=16−0=16>0（类空）。

改变坐标，C 改为 (−1,−1)，则 A 和 B 类空，B 和 C 间隔 sBC2=9−4=5>0（类空），但 A 和 C 的间隔 sAC2=9−9=0（类光）。

再改变坐标，C 改为 (−1,3)，则 A 和 B 类空，B 和 C间隔 sBC2=9−4=5>0（类空），但 A 和 C 的间隔 sAC2=1−9=−8<0（类时）。

因此，仅知道两对间隔类型，无法确定第三对的间隔类型。
