0. A Simple Review of Complex Analysis

0.1 Cauchy-Riemann Equations

As for defining whether a complex function is differentiable on the open set U, suppose that

(2)f(x+iy)=u(x,y)+iv(x,y)

where the functions u and v is both differentiable on the x−y plane. At a fixed z, let f′(z)=a+bi. f is differentiable means that for any point near z, the limit

(3)limΔz→0f(z+Δz)Δz

should be same. If we consider that Δz has an angle θ with the positive axis of x then the derivative of the function can be written as

(4)f′(z)=∂u∂xcosθ+∂u∂ysinθ+i(∂v∂xcosθ+∂v∂ysinθ)cosθ+isinθ

which should not depend on θ. So we can deduce a set of equations for any differentiable function, known as the Cauchy-Riemann equations:

(5)∂u∂x=∂v∂y
(6)∂u∂y=−∂v∂x

We can also write the differential as

(7)df=∂u∂xdx+∂u∂ydy+i∂v∂xdx+i∂v∂ydy

and any tantamount form using the Cauchy-Riemann Equations. It is not hard to find that as long as the real and imaginary part of f has continuous partial derivative on the real plane, they must be Harmonic Functions in order to be part of a holomorphic complex function. Both u and v need to be the solution to the 2 dimensional Laplace Equations:

(8)∂2u∂x2+∂2u∂y2=0
(9)∂2v∂x2+∂2v∂y2=0

 

0.2 Complex Calculus and Cauchy's Theorem

0.2.1 Complex Calculus

Given the fact that the definition of Riemann Sum can be extended to the complex field, Complex calculus can be simply defined as the Riemann Sum of the holomorphic function on the open set, which is not path-dependent, according to the properties of holomorphic functions, so typically we can use any preferred path to calculate an complex integral. For example, Let CR be the circle of radius R centered at the origin oriented counterclockwise. Let n be an integer. Then

(10)∫CR(z−a)ndz={0(n≠−1)2πi(n=−1)

Considering the fact that any holomorphic function defined on an open annulus can be represented by a Laurent series on that annulus, so integrals of holomorphic functions are reduced to integrals of polynomials.

0.2.2 Cauchy's Theorem

Cauchy's Theorem defines when a function's Calculus equals to zero on the annulus. It says that if a function f is holomorphic on a open set G―, then for the boundary C of G―, the Complex Calculus of f along that path equals to zero.

(11)∮Cf(z)dz=0

 

Proof.

  1. prove that for a simply connected region where f′ is continuous, the Cauchy's Theorem is right: Using the Green Theorem for the analytic functions, we can get

    (12)∮Cf(z)dz=∮C(udx−vdy)+i(udy+vdx)=∬S−(∂v∂x+∂u∂y)+i(∂u∂x−∂v∂y)dxdy=0

  2. prove that for a multiply connected region where f′ is continuous, the Cauchy's Theorem is right: Trivial considering cut every 'loop' in the region to the boundary and the region will be transformed into a simply connected region.

0.2.3 Two Important Lemmas

  1. If the function f is continuous in point a's neighborhood, given that the limit

(13)limz→af(z)(z−a)=k

exists, then the integral

(14)limδ→0∫Cδf(z)dz=ik(θmax−θmin)

where θmax and θmin are the boundaries of the path Cδ.

Proof. Suppose that the function can be expanded into Laurent Series around a

(15)f(z)=∑n=−∞∞an(z−a)n

and the i(θmax−θmin) is exactly

(16)limδ→0∫Cδ1z−adz

so the solution is trivial.

  1. Change the point a into the infinite point ∞. If

(17)limR→∞zf(z)=K

then

(18)limR→∞∫CRf(z)dz=iK(θmax−θmin)

where θmax and θmin are the boundaries of the path CR.

0.3 Laurent Series and Residue Theorem

0.3.1 Laurent Series

To calculate the complex calculus, we need to know the the asymptotic behavior near the singularity a. So suppose that the function can be expanded in to Series

(19)f(z)=∑n=−∞∞an(z−a)n

near the singularity a, in an annulus R1≤|z−a|≤R2. So the coefficients an can be denoted as

(20)an=12πi∮Cf(z)(z−a)n+1dz

where C is an arbitrary path in the annulus.

0.3.2 Residue Theorem

Now with the Laurent Series, we can change the Complex Calculus of arbitrary function f into the calculus of the polynomial according to

the Laurent Series of f.

Consider a path (counterclockwise) where f is analytic with some singular points inside, then we have

(21)12πi∮Cf(z)dz=∑k=1sRes(f,zk)

where Res refers to the value of integral along a small circle ( if zk is not the infinite point, choose the counterclockwise path, if zk is the infinite point, choose the clockwise path ) around the singular point zk , which is tantamount to the coefficient a−1 of the Laurent Series around the singular point.
